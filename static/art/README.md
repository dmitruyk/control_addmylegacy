# TV art slideshow images

Bundled 1920×1080 photographs for the full-screen TV gallery (city + nature themes).

| File | Theme |
|------|-------|
| `slides/city-*.jpg` | Urban / architecture |
| `slides/nature-*.jpg` | Landscapes / nature |

Source: [Lorem Picsum](https://picsum.photos/) — free for use in demos and prototypes. Replace with your own licensed collection in production if needed.

Manage visibility and order in Django admin → **Art slides**.
